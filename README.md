<h2>一、项目简介</h2>
基于SSM的校园心理健康网站的设计与实现

角色：管理员、用户

<h2>二、项目技术</h2>
前端 jsp、jquery、bootstrap，后端 maven、springmvc、spring、mybatis
<h2>三、功能介绍</h2>
<div class="markdown-heading" dir="auto">

<strong>学生</strong>
登陆注册、查询、收藏、评论、播放视频、更改个人信息、我的收藏、问卷调查、公告信息、查看分析结果、查看问卷答卷

<strong>管理员：</strong>
情况反馈管理、分析结果管理、问卷答卷管理、问卷调查管理、知识文章管理、文章类型管理、教师管理、学生管理、公告管理、轮播图管理、修改密码、个人信息
</div>
